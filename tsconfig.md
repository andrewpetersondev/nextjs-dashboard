# TsConfig Bases

## NOTES

1. since package.json uses `"type": "module"` I have to select `module` carefully.
    1. es2020
    2. es2022
    3. esnext
    4. nodenext

## Recommended

```json
{
  "compilerOptions": {
    "target": "es2016",
    "module": "commonjs",
    "esModuleInterop": true,
    "forceConsistentCasingInFileNames": true,
    "strict": true,
    "skipLibCheck": true
  },
  "display": "Recommended",
  "$schema": "https://www.schemastore.org/tsconfig"
}
```
